# FurLab Zones Tools Manual Checklist

Ручной чеклист для проверки редактора зон и связанных инструментов.

## Зоны

1. Выбор
- клик по зоне на сцене выбирает её;
- клик по зоне в дереве выбирает её;
- свойства справа переключаются правильно.

2. Удаление `base`-зоны
- базовую зону детали удалить нельзя;
- пункт меню выключен.

3. Удаление `split`-зоны
- удалить производную зону можно;
- при этом восстанавливается родительская зона.

4. Удаление `manual`-зоны
- зона, нарисованная вручную, удаляется полностью;
- не восстанавливает родителя;
- не зависает в дереве.

## Перо

5. Добавить точку
- на выбранной зоне клик по ребру добавляет новую вершину;
- вершина появляется в правильном месте.

6. Редактировать точку
- можно выбрать конкретную вершину;
- двигается именно выбранная вершина;
- слой точек не исчезает.

7. Граничная точка
- точка на контуре лекала не уходит в сторону;
- двигается только по границе детали.

8. Удаление внутренней точки
- внутренняя вершина удаляется `Delete/Backspace`;
- граничная не удаляется.

9. Сгладить кривую
- форма зоны реально меняется;
- это не только визуальный эффект.

10. Изменить кривизну
- вершина выбирается;
- ручки или изменение кривизны работают;
- форма перестраивается предсказуемо.

11. Общая вершина двух зон
- если две зоны делят одну и ту же вершину, она двигается связно.

12. Вершина на чужом ребре
- опасный случай не рвёт покрытие;
- либо блокируется, либо ведёт себя безопасно.

## Линия зонирования

13. Первая точка
- первый клик ставит явную первую точку;
- линия ещё не строится.

14. Вторая точка
- второй клик ставит вторую точку;
- после этого появляется пунктирная линия.

15. Совпадение с курсором
- точки ставятся под курсором без заметного смещения.

16. Подтверждение
- `Enter` реально делает split;
- после split в дереве появляются новые зоны.

17. Отмена
- `Esc` снимает draft split-line.

## Создание зон

18. Произвольный полигон
- точки ставятся;
- `Enter` или double click завершает зону;
- зона сохраняется.

19. Прямоугольник
- drag создаёт прямоугольную `manual`-зону;
- зона сразу появляется в дереве.

20. Эллипс
- drag создаёт эллиптическую `manual`-зону;
- зона сразу появляется в дереве.

## Горячие клавиши

21. `L`
- включает линию зонирования.

22. `X`
- включает добавить точку.

23. `V`
- включает редактировать точку кривой.

24. `C`
- включает изменить кривизну.

25. `S`
- включает сгладить кривую.

26. `R`
- включает прямоугольник.

27. `E`
- включает эллипс.

28. `Ctrl+Z / Ctrl+Y / Ctrl+Shift+Z`
- undo/redo реально работают по зонам.

## Интерфейс

29. Верхняя панель
- активен только один инструмент;
- flyout не подсвечивает вторую кнопку как активную.

30. Подменю инструментов
- иконки `20x20`;
- без рамок;
- шорткаты не вылезают;
- `Редактировать точку кривой` не переносится.

31. Редактор свойств зоны
- правая колонка значений стоит ровно;
- поле `Направление ворса, °` не уехало вправо;
- подписи не ломаются на две строки без необходимости.

32. Настройки отображения
- курсор `pointer`, не `text`;
- чекбоксы и подписи кликабельны нормально.

## Приоритетные риски

1. Удаление `manual`-зон в старых/смешанных состояниях.
2. Редактирование общей межзонной границы.
3. UX линии зонирования.
4. Горячие клавиши после недавних правок.
